# Remotion Videos

## Installation


Install all dependencies using npm:

```sh
npm i
```

Copy the script into your bin folder:

```sh
cp remotion-videos ~/.local/bin/remotion-videos
```

## Usage

Just run the script with the video you want to use. The video should be named after the title you want to use.

```sh
remotion-videos <video-file>
```

Afterward, you will be asked to select a format and a style. The video will then be rendered and copied next to the original video file.
